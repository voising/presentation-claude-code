C'est quoi Claude Code ?

C'est comme ChatGPT.

lance des scripts,
installe des apps.

Un développeur prend le contrôle de votre ordinateur.

$ curl -fsSL https://claude.ai/install.sh | bash

Mac, Linux, Windows (WSL). C'est tout.

01
Mon process iOS

Le PRD

Ouvrir ChatGPT ou Gemini. Coller ce prompt :

"Let's build a PRD.md document together. You'll ask me only one relevant question at a time. No summary, just one question each time. We're building an iOS app that will [APP_TOPIC]."

Il vous interroge, vous répondez. À la fin : un cahier des charges complet.

02
Mon process iOS

Le design system

Dans Claude Code, avec le PRD en contexte :

"Build a design system with all the necessary components for this app @prd.md."

Il pose les fondations visuelles avant d'écrire la moindre feature.

03
Mon process iOS

L'icône

Dans Claude Code, un prompt méta :

"Donne-moi un prompt que je vais coller dans Gemini pour créer une icône d'application iOS."

Claude Code connaît l'app — il écrit le prompt parfait pour Gemini. On copie, on colle, on a l'icône.

04
Mon process iOS

Tout construire

Et ensuite, ce prompt — qui change tout :

"Please build the complete product. Stop only when the app is fully built, when you covered 100% of the features."

Il avance, vérifie, recommence, jusqu'à ce que l'app soit finie.
